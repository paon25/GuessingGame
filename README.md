# GuessingGame 🎮

## Description
Jeu de devinettes en Java où l'utilisateur doit deviner 
un nombre aléatoire entre 1 et 100 en 10 essais maximum.

## Fonctionnalités
- Génération d'un nombre aléatoire entre 1 et 100
- 10 essais maximum
- Indications si le nombre est trop grand ou trop petit
- Message de victoire ou défaite à la fin

## Comment exécuter
1. Cloner le projet : `git clone https://github.com/paon25/GuessingGame.git`
2. Ouvrir dans IntelliJ IDEA
3. Exécuter la classe Main.java
